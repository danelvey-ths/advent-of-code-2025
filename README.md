# Advent of Code 2025

Solutions for Advent of Code 2025 (12 days).

## Structure

```
.
├── inputs/          # Puzzle inputs (one file per day)
├── solutions/       # Solution scripts (part1.py and part2.py per day)
└── README.md
```

## Running Solutions

Each day has two parts in its own folder:

```bash
# Run from VSCode or terminal
python solutions/day01/part1.py
python solutions/day01/part2.py
```

## Inputs

Place your puzzle inputs in `inputs/day01.txt`, `inputs/day02.txt`, etc.
